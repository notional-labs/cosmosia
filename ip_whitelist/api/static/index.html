set_config

curl -X POST http://127.0.0.1:5001/set_config \
-H 'Content-Type: application/json' \
-H 'Authorization: Basic dGVzdF9wYXNzd29yZA==' \
-d '{"data":"2.2.2.2/24 3.3.3.3"}'

data is list of IP or CIDR separated by white space.
dGVzdF9wYXNzd29yZA== is base64 encoded of your password. You could get it by:
echo -n 'test_password' | base64

A successful request still returns error or nothing as the server being restarted/reloaded immediately.
